CSS Selector

Hay muchos tipos diferentes de selector CSS que le permiten orientar las reglas a elementos especificos en un documento HTML

Los selectores CSS distinguen entre mayusculas y minusculas, por lo que deben coincider excatamente con los bombres de elemento y los valores de atributo

Tipos de selectores
