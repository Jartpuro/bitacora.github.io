Text Properties

by TPS SENA

Este ejemplo combina muchas de las tenicas mostradas en los ejercicios

Hay propiedades para controlar el tipo de fuente, el tamaño, la ubicación, el estilo y el espaciado

Caracteristicas

Puede controlar el espacio entre las lineas de texto, las letras individuales y las palabras. El texto tambien se puede alinear a la izquierda, derecha, centrada o justificada. Tambien puede tener sangría
