Letter and Word Spacing

by TPS SENA

El kerning es el término que utilizan los tipografos para el espacio entre cada letra. Puede controlar el espacio entre cada letra con la propiedad: letter-spacing. Es particularmente util para aumentar el kerning cuando su encabezado o frase esta en mayusculas. Si el texto está en un caso o normal, aumentar o disminuir el kerning puede dificultar la lectura.

El intervalo predeterminado

Cuando se especifica un valor para estas propiedades, se debe dar en EMS. El intervalo predeterminado entre las palabras se establece por el tipo de letra(a menudo alrededor de 0.25 EM), y es poco probables que deba cambiar esta propiedad con regularidad.
