En HTML% no es necesario proporcionar valores para todos los atributos, como los controles, reproduccion automatica y atributos de blucle utilizados con el elemento <video<. Estos atributos son como interruptores de encdencido/apagado. Si el atributo esta presente, activa es opcion. Si se omite el atributo, la opcion esta desactivada. Si el navegador no admite el elemento <video> o el formato de video utilizado, se mostrará lo que esté entre la <video< de apertura y las etiquetas de </video> de cierre

preload: este atributo le indica al navegador que hacer cuando se carga la pagina. Puede tener uno de tres valores:
none:El navegador no debe cargar el video hasta que el usuario presiona Play.
auto: el navegador debe descargar el video cuando se carga la pagina.
metadata: el navegador debe simplemente recopilar informacion con el tamaño, el primer fotograma, la lista de pistas y la duración
